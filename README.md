# Devansh Jaiswal - Portfolio

A personal portfolio website for Devansh Jaiswal, an AI Developer and BCA Artificial Intelligence student at Jain University. 

## Features
- **Modern UI**: Dark-themed, glassmorphism design with responsive layouts.
- **Dynamic Animations**: Smooth transitions, typewriter effects, and micro-interactions.
- **Sections**: Home, About, Technical Skills, Featured Projects, and Contact.

## Technologies Used
- HTML5
- CSS3 (Vanilla CSS, custom properties, animations)
- JavaScript (DOM manipulation, event handling)

## How to Run
This is a static website. You can run it by simply opening the `index.html` file in any modern web browser. No local server or build tools are required.

## Structure
- `index.html`: Main HTML document.
- `styles.css`: All styling and animations.
- `script.js`: Interactive logic (typewriter effect, mobile menu, smooth scrolling).
- `assets/`: Contains images used in the project.
